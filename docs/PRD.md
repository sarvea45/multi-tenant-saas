# Product Requirements Document
## User Personas
1. **Super Admin:** System owner, manages all tenants.
2. **Tenant Admin:** Customer, manages their own projects/users.
3. **User:** Employee, works on tasks.

## Functional Requirements
1. System shall isolate data between tenants.
2. System shall allow login via Subdomain + Email.
3. System shall prevent users from seeing other tenants' projects.
